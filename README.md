# EthControl
Mining rig controlling remotely

Register your account today @ https://ethmonitoring.com

- Excavator
- SGMiner
- BMiner
- PhoenixMiner
- Claymore Dual miner

- XMR-Stak
- CCMiner (Forks also)
- Gateless gate
- NoncerPro
- TRex
- XMRig
- TeamRedMiner
- TTMiner

# Requirements

- This tool required .NET Framework 4.6.1 installed (https://www.microsoft.com/en-us/download/details.aspx?id=49981)
